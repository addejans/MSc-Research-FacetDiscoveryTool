# Notebooks

The notebooks in `notebooks/experiments/` are preserved as historical research artifacts from the MSc facet-classification work.

They are exploratory, chronological, and not intended as production code. The useful research idea is the workflow:

1. load projected Fourier-Motzkin inequality outputs;
2. normalize and map inequalities into feature representations;
3. cluster/classify like-facets;
4. infer repeated facet families across parameterized constraint instances.

Start with `Classification_Attempt1.ipynb` for the clearest ML/classification-oriented notebook.
