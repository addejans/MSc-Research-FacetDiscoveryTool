# Data

This folder is for raw and derived data used by the historical facet-classification notebooks.

The repository still contains several historical Fourier-Motzkin output directories at the top level, including `bounds/`, `k0/`, `sorted20180211/`, `sorted20171105_Reduced/`, and `sorted_reduced/`. Those folders are preserved because the notebooks may reference them by their original paths.

`data/raw/` contains loose raw files that were previously at the repository root.
